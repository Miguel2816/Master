Carpeta creada para alojar los informes/resultados del análisis de sentimiento que se ha generado sobre:

- Datos Limpios: set de datos que se ha preprocesado totalmente.
- Datos con emoticonos y emojis: set de datos que se ha preprocesado igual que el anterior pero sin incluir los emojis, emoticonos y signos de puntuación, ya que las herramientas usadas para el análisis de sentimiento son capaces de capta el sentimiento de estos tres elementos sin necesidad de preprocesarlos.
