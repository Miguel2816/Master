Carpeta creada para alojar los notebooks generados:

- 00-Testint-Things: notebook orientado a hacer pruebas.
- 01-Get-Twitter-Data: notebook cuyo objetivo es conseguir el dato de twitter y guardarlo.
- 02-Join-Files: notebook creado para juntar todos los archivos que se han ido almacenando.
- 03-Analyze-Data: notebook diseñado para analizar el dato, entenderlo e investigar.
- 04-Clean-Data: notebook generado para hacer la limpieza del dataset.
- 05-Sentiment-Analysis: notebook creado para realizar el análisis de sentimiento sobre el dataset totalmente preprocesado.
- 06-Sentiment-Analysis-2: notebook creado para realizar el análisis de sentimiento sobre el dataset  preprocesado igual que el anterior, pero sin limpiar ni los emoticonos ni los emojis ni los signos de puntuación.
- 07-Clusterization: notebook diseñado para realizar los embeddings y la búsqueda de clusters y patrones en el corpus.
- 08-tfm-bert-model: notebook creado en Google Colab para poder generar el embedding a partir del algoritmo BERT.
- 09-Análisis-Dataset-Kaggle: notebook generado para analizar los datos de un dataset de tweets recogidos sobre aerolíneas estadounidenses clasificados manualmente.
